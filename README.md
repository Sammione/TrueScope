# TrueScope: ESG Greenwashing Detector

This is an evidence-first ESG assistant that can:
- Upload an ESG report (PDF/text)
- Extract ESG claims (targets, metrics, certifications)
- Verify claims against report context + optional external evidence uploads
- Output verdicts (supported / weak / unsupported / contradictory) + a greenwashing score
